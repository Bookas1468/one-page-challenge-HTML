# one-page-challenge-HTML
one page html challenge


its one page with the only libraries being jquery and bootstrap

text boxes will appear, press that key (either on your keyboard or on the on-screen one) to make it disapear before it turns fully red to earn a point.
if it turns red before you hit that key you loose a point. you can only press the key of the first item (top-left), the other ones wont dissapear even if you press their key.

you can change fonts and difficulty in the header
